Detailed Description

template<char FILL_CHAR = '#', char EMPTY_CHAR = '-'>
class utils::utils::utils::print::Progressbar< FILL_CHAR, EMPTY_CHAR >

A class representing a Progressbar. Use this to indicate progress being made in a process or loop.

Printing the progressbar is not threadsafe.

e.g.

utils::print::Progressbar bar(list.size()); for (auto val : list) { process(val); utils::Logger::GetConsoleStream() << ++bar; } utils::Logger::GetConsoleStream() << bar.done();

or

utils::print::with_progressbar(list, utils::Logger::GetConsoleStream(), process);
