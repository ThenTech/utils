T  from = std::numeric_limits<T>::min(),
T  to = std::numeric_limits<T>::max() 
)
static

Generate a container (default: vector<T>) with amount items, randomly distributed between from and to.

Only fundamental data types are supported (numbers, chars and bools). These are randomly chosen between between from and to, while booleans are chosen with a 1/2 probability.

To generate booleans with other probabilities, use generate_bool, to generate a string (instead of vector) when choosing chars, use generate_string.

For other data types, the function will not be enabled.

Parameters
amountThe amount of items to generate.
fromThe lower bound for the distribution. (default: min T)
toThe upper bound for the destribution. (default: max T) If upper < lower, these are swapped.
Returns
Returns a new Container containing amount items of type T.
