◆ shift_signed()

template<class T , size_t Tlen = utils::bits::size_of<T>()>
ATTR_MAYBE_UNUSED static ATTR_NODISCARD T utils::bits::shift_signed ( const uint64_t  value,
const size_t  src_bits 
)
inlinestatic

Create a signed value of type T with the given value that is only src_bits long.

This abuses the fact that right-shifting a signed number with an MSB of 1, will fill in the new MSBs as 1, making the original value signed in the new (bigger) size.

Template Parameters
TTarget type (signed).
Parameters
valueThe value to make signed in the available bits of T.
src_bitsThe amount of bits that are used in value.
Returns
Returns a signed or unsigned number, depending on whether value[src_bits : 0] was considered signed.
