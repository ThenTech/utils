Detailed Description

template<typename ... Types>
struct utils::algorithm::any_tie< Types >

The any_tie struct. Use to tie a result back into itself for any type that supports structured bindings access.

Example: any_tie(value1, value2) = value2;

because this does not work: auto& [value1, value2] = value2;

Structured bindings uncovered - Dawid Zalewski [ C++ on Sea 2020 ]
