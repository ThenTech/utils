Calculate value raised to the power Exp, where Exp must be an unsigned int.

The result is calulated by unpacking value times itself Exp times.

Template Parameters
ExpThe power to raise to as template parameter. Default is squared.
Parameters
valueThe value to raise.
Returns
Returns the result of Exp times multiplying \value with itself.
