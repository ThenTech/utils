template<class T, class U> \

template<class T, class U> \
using has_ ## F = can_apply<dot_ ## F ## _r, T, U>; \
template<typename T, class U> \
inline constexpr bool has_ ## F ## _v = has_ ## F<T, U>::value;

Creates a has_<F>_v function that returns a boolean ::value to indicate if the given object has the function F.

e.g. CREATE_HAS_FUNCTION(find) creates: dot_find_r<T, U> that calls T.F() has_find that checks if dot_find can be applied on T has_find_v that returns has_find::value
