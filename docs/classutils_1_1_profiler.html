Detailed Description

The Profiler class to provide Chromium performance trace json output. Use chrome://tracing/ to load these files and see a timeline with the starts and durations of scopes.

With UTILS_PROFILE_BEGIN_SESSION(<file name>), a new trace file will be made. End a session with UTILS_PROFILE_END_SESSION() or by exiting the scope. Best begin a session from main().

Add UTILS_PROFILE_FUNCTION() at the start of a function, or UTILS_PROFILE_SCOPE(<name>) within a nested scope to trace its walltime.

Returning from the (function) scope will cause the scoped ProfileTimer to write the walltime to the trace file.
