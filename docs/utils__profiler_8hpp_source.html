21  * \brief The Profiler class to provide Chromium performance trace json output.
22  * Use chrome://tracing/ to load these files and see a timeline with the
23  * starts and durations of scopes.
24  *
25  * With UTILS_PROFILE_BEGIN_SESSION(<file name>), a new trace file will be made.
26  * End a session with UTILS_PROFILE_END_SESSION() or by exiting the scope.

29  * Add UTILS_PROFILE_FUNCTION() at the start of a function, or
30  * UTILS_PROFILE_SCOPE(<name>) within a nested scope to trace its walltime.
