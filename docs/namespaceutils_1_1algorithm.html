Check if the given container contains the given item. The type of item must be something that the container holds. e.g. vector<int> <- int string <- char and not: vector<Node> /- int.

Wrap CPPItertools as ::iter Reference: https://github.com/ryanhaining/cppitertools (04/08/2019)

Parameters
containerThe container to look in.
itemThe item to look for.
Returns
Returns an std:optional<size_t> holding the offset from std::begin(container) if the item was found, or nothing if not.
